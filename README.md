## What is Ledger Finance?

Ledger Finance is a complete business accounting application contained in a **single HTML file**. No installation, no server, no database, no account, no subscription. Just download the file, double-click it, and start tracking your finances.

All data is stored locally in your browser's localStorage — nothing is ever sent to a server.

---

## ✨ Features

### 📊 Dashboard
- Live overview of income, expenses, net profit, and pending invoices
- Running balance banner showing starting balance, net activity, transfers, and current balance
- Recent transactions feed
- Expense breakdown by category with visual bar charts
- Transfer breakdown with in/out totals

### 💳 Transaction Tracking
- Log **income**, **expenses**, **transfers in**, and **transfers out**
- Running balance column on every transaction
- Filter by transaction type
- Edit or delete any transaction after entry
- Custom categories — create your own and they save permanently

### 📄 Invoice Management
- Create invoices with client name, description, amount, and due date
- Track payment status (Pending / Paid)
- One-click mark as paid or revert to pending
- Filter by status

### 📈 Financial Reports
- Profit & Loss statement with starting balance and current balance
- Monthly income vs expense trend chart
- Income sources breakdown
- Invoice summary (paid vs outstanding)

### ⚙️ Settings & Data
- Set a starting balance for accurate running totals
- Export all data to a named JSON backup file
- Import from any previous backup
- All data stored locally — 100% private

## 💾 Data & Privacy

- All data is stored in **browser localStorage** on your device
- **Nothing is sent to any server** — ever
- To back up your data: click **↓ Export** in the sidebar
- To restore your data: click **↑ Import** and select your backup file
- To move data to another device: export on the old device, import on the new one

> **Tip:** Export a backup regularly and save it to Google Drive, Dropbox, or a USB drive for safekeeping.
